# Resumo
As extensões do Modelo ER são utilizadas para aumentar o poder de expressão do modelo, provendo novos meios de representações. Uma dessas extensões é a abstração de dados, que omite detalhes estruturais para facilitar a visão global do tópico em questão. As principais extensões do Modelo ER incluem herança de atributos, especialização, generalização, agregação e outras. A especialização é o processo de definir uma entidade que é um subconjunto de outra entidade, enquanto a generalização é o processo de definir uma entidade que é um superconjunto de outra entidade. A herança de atributos permite que os atributos dos conjuntos de entidades de nível superior sejam herdados pelos conjuntos de entidades de nível inferior. Além disso, existem conceitos como participação total ou parcial e exclusão mútua ou sobreposição, que são utilizados para descrever as relações entre as entidades. A agregação é uma extensão que permite que um conjunto de relacionamentos seja considerado como uma entidade com o objetivo de participação em outros relacionamentos.

## Conceitos importantes
* **Herança de Atributos**: os atributos dos conjuntos de entidades de nível superior são herdados pelos conjuntos de entidades de nível inferior.
* **Especialização**: é o processo de definir uma entidade que é um subconjunto de outra entidade.
* **Generalização**: é o processo de definir uma entidade que é um superconjunto de outra entidade.
* **Agregação**: permite que um conjunto de relacionamentos seja considerado como uma entidade com o objetivo de participação em outros relacionamentos.
* **Participação Total ou Parcial**: descreve a relação entre as entidades, indicando se a união de todos os conjuntos de entidades específicos resulta no conjunto de entidades genérico.
* **Exclusão Mútua ou Sobreposição**: descreve a relação entre as entidades, indicando se um conjunto de entidades genérico pode participar de um e apenas um dos conjuntos de entidades específicos.

## Pontos que merecem atenção
* A herança de atributos é fundamental para entender como as entidades se relacionam entre si.
* A especialização e a generalização são conceitos importantes para definir as relações entre as entidades.
* A agregação é uma extensão útil para representar relacionamentos complexos entre as entidades.
* A participação total ou parcial e a exclusão mútua ou sobreposição são conceitos importantes para descrever as relações entre as entidades.
